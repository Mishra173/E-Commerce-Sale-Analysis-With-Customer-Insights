# E-Commerce-Sale-Analysis-With-Customer-Insights (End-to-End Project)

##  Project Overview

This project presents an end-to-end analysis of e-commerce sales data to uncover revenue trends, customer behavior, and regional performance.

The workflow includes:
- Data Cleaning & Exploratory Data Analysis (EDA) using Python
- Business Analysis using SQL
- Interactive Dashboard Development using Power BI



##  Tools & Technologies

- Python (Pandas, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- DAX
- Data Modeling
- Data Visualization



##  Data Cleaning & EDA (Python)

- Handled missing values and inconsistent data
- Performed data transformation using Pandas
- Conducted exploratory data analysis
- Created revenue trends, distribution plots, and correlation analysis using Matplotlib & Seaborn


##  SQL Analysis

Performed business-focused analysis using SQL:

- Total Revenue & Order Analysis
- Monthly Revenue Trends
- Top Products & Top Customers
- Country-wise Revenue Analysis
- Customer Ranking
- Repeat Customer Identification
- Generated actionable insights and recommendations

---

## 📊 Power BI Dashboard Features

- Total Revenue & YTD KPI
- Total Orders & Total Customers KPI
- Monthly Revenue Trend
- Revenue vs Orders Comparison
- Top 5 Products by Revenue
- Top 5 Customers by Revenue
- Country-wise Revenue Map
- Interactive Slicers (Date, Country, Year)



##  Key Insights

- Revenue shows seasonal fluctuation with stronger Q4 performance.
- Europe contributes the highest revenue share.
- Top 5 products and customers drive significant revenue concentration.
- Revenue growth aligns closely with order volume trends.
- High-value customers contribute a large share of total revenue.


##  Business Recommendations

- Focus marketing efforts during low-performing months.
- Retain high-value customers through loyalty strategies.
- Promote top-performing products for cross-selling.
- Diversify the customer base to reduce revenue dependency risk.
